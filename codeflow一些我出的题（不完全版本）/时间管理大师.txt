时间管理大师

c++同学，一枚广金萌新，怀揣着对大学美好生活的向往，大一就加入了许多个组织社团。在某个周六这一天刚好有n个活动要参加，c++是一名优秀的时间管理大师，能完美处理好使n个活动不冲突。但是c++虽然优秀但是本性爱玩，喜欢在周六这一天花连续的s分钟在宿舍追剧，一天却只有 m分钟，还有活动时间占用着。

假设每个活动的时间段为(l,r)min，且不冲突，并且在这段时间里，c++很忙，是不能呆在宿舍追剧。当给定所有 n个时间间隔，请问c++当天可以在宿舍追剧吗？


输入：
第一行整数 t（1≤t≤$10^2$），代表测试用例数
每个测试用例的第一行包含三个整数 n 、s和 m（1≤n≤$10^4$，1≤s,m≤2*$10^5$），即c++的活动数、在宿舍追剧的时间以及一天的分钟数。
接着是 n行，其中每一行包含两个整数 l和r(0≤l≤r≤m)，代表每个活动的时间间隔，每个活动不冲突，且保证输入的后一个时间段l比前一个时间段r大。

输出：
对于每个测试用例，如果 c++可以在宿舍追剧输出 "Yes"(不带引号)，否则输出 "No"(也不带引号)。





解题：
找出是否有一个长度至少为 s的空闲时间间隔。
空闲时间间隔：每段活动的时间差，还有第一段活动前的时间，最后一段活动后的时间。
#include<bits/stdc++.h>
using namespace std;
#define int long long
signed main()
{
	int t;
	cin>>t;
	while(t--)
	{
		int n,s,m;
		cin>>n>>s>>m;
		int a[n][2];
		for(int i=0;i<n;i++)
		{
			cin>>a[i][0]>>a[i][1];
		}
		int flag=0;
		if(a[0][0]>=s || m-a[n-1][1]>=s)
		{
			flag=1;
		}
		else
		{
			for(int i=1;i<n;i++)
			{
				if(a[i][0]-a[i-1][1]>=s)
				{
					flag=1;
					break;
				}
			}
		}
		if(flag==1)
		{
			cout<<"Yes"<<endl;
		}
		else
		{
			cout<<"No"<<endl;
		}
	}
}
