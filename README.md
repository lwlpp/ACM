# codeflow

**一个广金周赛举办平台**，目的是面向广金校内小白的蓝桥杯培训和算法学习，地址：[oj.jdygpt.com](https://oj.jdygpt.com/problem)，还有公众号：CodingFlow，B站：CodeFlow，我在其中担任出题人和讲题人之一哦

宣传语：

🔥欢迎大家成为【CodeFlow】的成员----一个为大学生量身打造的算法社区！在这里，我们一起探索代码的魅力，提升算法能力，结交志同道合的伙伴。无论你是算法小白，还是已经有了编程基础，我们都欢迎你成为【CodeFlow】的成员，和我们一起成长！💪🏻

🎉【组织介绍】

CodeFlow是一个帮助广大学子学习算法和编程的平台，在这里你可以学习算法、刷题，享受Coding的乐趣, 与同样对算法感兴趣的朋友一起交流，我们会邀请优秀算法竞赛大咖、老师以及同学帮助大家学习算法、编程、前沿技术，与大家一起共同进步，至未来！

📌【你可以获得】

1.系统的算法学习资源，包括视频教程、书籍推荐、题库分享等
2.与各类编程高手和同龄人交流学习的机会，扩展你的人脉圈
3.参与内部编程挑战赛，检验和提升你的编程能力
4.专业指导老师和学长学姐的经验分享，帮助你走好编程路上的每一步
5.丰富的实践项目机会，提升你在真实项目中的编程应用能力
